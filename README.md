# Codeforces Competitive Programming - Python

This repository contains my solutions to Codeforces problems in Python.  

## Structure

- **`problemset/`** – Solved individual problems organized by difficulty.  
- **`contests/`** – Solutions organized by contest.  

## Purpose

This repo is mainly for tracking my own progress in competitive programming, but it may also help others who want to see Python solutions for Codeforces problems.
